《首页 管理员接口:

  • 管理员登录
  • 邮箱登录
  • 添加管理员
  • 添加多个管理员

管理员登录

method：post
url: /api/v1/admin/login

参数列表

参数名类型必需描述
namestringtrue登录名
passwordstringtrue密码

返回结果: Android Code | IOS Code

{
  "userId": "string //用户id",
  "userName": "string //用户名",
  "password": "string //密码"
}

邮箱登录

method：post
url: /api/v1/admin/emailLogin

参数列表

参数名类型必需描述
emailstringtrue
passwordstringfalse

返回结果: Android Code | IOS Code

{
  "userId": "string //用户id",
  "userName": "string //用户名",
  "password": "string //密码"
}

添加管理员

method：post
url: /api/v1/admin/add

参数列表

{
  "name": "string //名字",
  "password": "string //密码"
}

返回结果: Android Code | IOS Code

{
  "userId": "string //用户id",
  "userName": "string //用户名",
  "password": "string //密码"
}

添加多个管理员

method：post
url: /api/v1/admin/addMany

参数列表

[
  {
    "name": "string //名字",
    "password": "string //密码"
  }
]

返回结果: Android Code | IOS Code

[
  {
    "userId": "string //用户id",
    "userName": "string //用户名",
    "password": "string //密码"
  }
]
